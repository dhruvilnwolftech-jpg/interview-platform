## 🚨 Troubleshooting

### "Can't connect to backend"
- Check internet
- Refresh page
- Backend at https://interview-platform-bdot.onrender.com/ should work

### "Session code not working"
- Code must match exactly (case-insensitive)
- Support person must create first
- Confirm session is active

### "Buttons not responding"
- Hard refresh: Ctrl+Shift+R
- Try different browser
- Check browser console (F12)

### "No data showing"
- Wait 2-3 seconds for API response
- Check internet connection
- Try clicking "Load Sessions" again
